Secuencia de Escape

\'' = Mostrar doble comilla dentro de una cadena en la salida de información
\\ = Barra invertida, diagonal inversa, es para mostrar un back slash
\n = Esto es un salto de linea
\r = Retorno
\t = Tabulacion
\b = Retroceso
\f = Salto de pagina
